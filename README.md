# Mervv - Özel Site

Bu proje, özel bir web sitesi için basit bir başlangıç noktasıdır.

## Kurulum

1. Bu klasörü VS Code ile açın.
2. `app.js` dosyasını açın ve Gemini ile oluşturduğunuz kodu buraya yapıştırın.
3. `index.html` dosyasını tarayıcıda açarak sitenizi görüntüleyin.

## Dosyalar

- `index.html`: Ana sayfa.
- `style.css`: Stil dosyası (romantik tema).
- `app.js`: JavaScript kodlarınızın bulunacağı dosya.
